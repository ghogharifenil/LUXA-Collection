
# 👗 LUXA Collection

LUXA Collection is a modern Django-based fashion e-commerce platform designed for a seamless shopping experience. Customers can browse products, manage carts, place orders, track purchases, and contact support, while administrators can manage products, orders, and customer interactions through a dedicated dashboard.

---

## ✨ Features

### Customer Features
- User Registration & Login
- Browse Fashion Products
- Product Categories
  - Saree
  - Dress
  - Shirt
  - Pant
- Product Details Page
- Add to Cart
- Wishlist Support
- Place Orders
- Order History
- Invoice Generation (PDF)
- Email Notifications
- Contact Support

### Admin Features
- Secure Admin Login
- Dashboard Overview
- Add Products
- Manage Products
- View Orders
- Update Order Status
- Customer Message Management
- Send Customer Support Updates via Email
