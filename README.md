
# 👗 LUXA Collection

LUXA Collection is a modern Django-based fashion e-commerce platform designed for a seamless shopping experience. Customers can browse products, manage carts, place orders, track purchases, and contact support, while administrators can manage products, orders, and customer interactions through a dedicated dashboard.

---

## ✨ Features

### Customer Features
- User Registration & Login
- Browse Fashion Products
- Product Categories
  - Saree
  - Dress
  - Shirt
  - Pant
- Searching Product 
- New Collection Exploler
- Product Details Page
- Add to Cart
- Place Orders
- Order Confirm Mail 
- Email Notifications
- Contact Support By LUXA Admin 
- About LUXA

### Admin Features
- Secure Admin Login
- Dashboard Overview
- Add Products (Listing Page)
- Add New Product is New section Feture
- Manage Products
- View Orders
- Update Order Status
- Customer Message Management
- Send Customer Support Updates via Email
